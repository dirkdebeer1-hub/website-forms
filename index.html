Installation Instructions

  1. Create a new repository on GitHub with a README file
  2. Upload this HTML file as index.html to your repository
  3. Go to Settings → Pages → Select "Deploy from branch" → Choose "main" branch
  4. Your dashboard will be available at: https://[username].github.io/[repository-name]

Note: It may take a few minutes for your site to be published after enabling GitHub Pages.
